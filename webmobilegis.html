Gamaku WebGIS: Road and Building Information System for Universitas Gadjah Mada

This project was created to help solve the issue of manual and unorganized infrastructure monitoring across UGM’s campus. I built WebGIS Gamaku, an interactive map platform that brings together data on roads and buildings, allowing users to visualize, manage, and report facility conditions in real time.
